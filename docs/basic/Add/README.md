# Git Adding

## - Add all file into repo
```bash
git add .
```


## Adds content from all *.txt files under **docs** directory and its subdirectories:
```bash
git add docs/\*.txt
```

## Add All Files of Multiple Extensions
```bash
git add *.js *.css *.html
```

## Add Files Recursively in All Subdirectories
```bash
git add '**'/*.js '**'/*.css '**'/*.html
```
